# Admin Shop -

## Description
Le module **Admin Shop** est le pilier économique Ce système a été conçu pour garantir une économie stable, régulée et performante, offrant aux joueurs un accès constant aux ressources essentielles sans dépendre de l'inflation du marché des joueurs.

## Fonctionnalités Clés
*   **Stabilité Économique :** Prix fixes et contrôlés pour éviter l'hyper-inflation.
*   **Disponibilité Permanente :** Stock illimité sur tous les objets listés.
*   **Transactions Instantanées :** Interface optimisée pour un traitement rapide et sécurisé des échanges.
*   **Intégration Optimisée :** Développé pour fonctionner sans latence au sein de l'infrastructure de Phara-Hosting.

## Pourquoi ce système ?
Contrairement aux shops de joueurs, l'Admin Shop sert de référence pour la valeur réelle des items. Il garantit que chaque nouveau joueur peut progresser dans le jeu de manière autonome et équitable, tout en maintenant la valeur de la monnaie du serveur sur le long terme.

---

## Configuration & Usage
*   **Accessibilité :** Utilisable via l'interface `/shop` ou via les PNJ désignés au spawn.
*   **Maintenance :** Les prix sont ajustés périodiquement par l'administration via la configuration YAML.
